AetherMesh Web Flasher

Flash a Heltec V4 node over USB, straight from your browser — no install needed.

This browser can't flash over USB.
The Web Serial API is only available in desktop Chrome, Edge, or Opera. Please open this page in one of those. (Firefox, Safari, and most mobile browsers aren't supported.)

1. Choose firmware

Use the aethermesh-heltec-<hash>-usb.bin image (the complete USB build with bootloader), not the -ota.bin.

2. Put the Heltec in download mode

  1. Hold the PRG (BOOT) button.
  2. While holding it, briefly press and release RST.
  3. Release PRG. The screen stays blank — that's correct.

This avoids a mid-flash USB disconnect these boards are prone to.

3. Flash

After a successful flash, tap RST once and the node boots the new firmware. Nothing here is destructive to your other nodes — this only touches the one board you connect.

Prefer no cable at all? Once a node runs OTA-capable firmware, use the app's Settings → Firmware Update to flash over Bluetooth. This web flasher is for the first-time USB bootstrap and recovery.
