nRF52 Bootloader Flow

The RAK4631 uses a USB Mass Storage bootloader. Double-tap RST to enter bootloader mode, then drag and drop the firmware:

  1. Double-tap the board's RST button quickly.
  2. A USB folder named RAK4631 or NORDIC will mount on your computer.
  3. Download the latest firmware binary using the link below.
  4. Drag and drop the downloaded .uf2 file onto the mounted folder.
